# 🚀 SCOREPION — AI ERA SURVIVAL & DOMINANCE IMPLEMENTATION PROMPT

## Role
You are the **Principal Product Architect + Growth Systems Engineer + UX Strategist**.

You are working inside Replit to implement structural upgrades to Scorepion.

Your mission is NOT to polish UI.

Your mission is to transform Scorepion from:

> A clean football prediction app

into:

> A habit-forming, socially tense, defensible sports competition platform that survives in the AI era.

You must implement systemic upgrades, not cosmetic tweaks.

---

# 🧠 CONTEXT: STRATEGIC THREATS

Scorepion faces these structural risks:

1. Low switching cost.
2. No daily obligation loop.
3. Engagement depends on external match importance.
4. Leaderboard lacks emotional tension.
5. Groups are static (no live social feed).
6. Prediction is mechanical, not strategic.
7. No personalization layer.
8. No seasonal narrative.
9. No defensible moat against AI pick bots.
10. No strong outcome positioning (status, identity).

You must solve these at architecture level.

---

# 🎯 PRIMARY OBJECTIVE

Make Scorepion:
- Habit-forming
- Socially competitive
- Emotionally tense
- AI-enhanced (not AI-replaced)
- Scalable across leagues and seasons

---

# 🏗 PHASE 1 — CORE RETENTION ENGINE

## 1️⃣ Implement Daily Picks Pack System

Create a new backend + frontend module:

### Daily Picks Pack:
- Auto-generate 3–7 matches per user per day
- Based on:
  - Followed leagues
  - Group league overlap
  - Match importance
  - Kickoff proximity
  - Table impact
- Track:
  - Completion status (x/y predicted)
  - Countdown to first kickoff
  - Streak continuity

### Database additions:
- daily_pack table
- user_daily_progress table
- streak_tracking table

### UX:
- Top banner on home:
  - “Today’s Picks”
  - Progress bar
  - Streak badge
  - CTA: “Finish Today’s Picks”

Daily streak only increments when pack completed.

This is mandatory.

---

# ⚔️ PHASE 2 — DIRECTED COMPETITION

## 2️⃣ Implement Chase Mode

### Backend:
For each leaderboard:
- Assign each user a “chase target” (rank above)
- Calculate:
  - Points gap
  - Matches that can flip rank
  - Potential swing impact

### Frontend:
Leaderboard must show:
- Your pinned card
- “You’re 38 pts behind Tom”
- Visual gap indicator
- Rank swing preview on upcoming matches

Add:
- Rank change animation
- Weekly reset countdown

Leaderboard must create tension.

---

# 🔥 PHASE 3 — STRATEGIC PREDICTION LAYER

Prediction must not be a simple score entry.

## 3️⃣ Add Strategy Mechanics

Implement:

### 1 Boost Pick per Day:
- User selects 1 match
- Points doubled
- Cannot change after lock

### Optional Upset Bonus:
- If picking underdog win
- Additional points if correct

### After Lock:
Transform match view into:
- Live prediction tracker
- Points at stake
- Rank impact preview

Prediction becomes a game of risk.
